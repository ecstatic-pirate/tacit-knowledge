# User Flows

## 1. Create Campaign

```
Start → Enter expert details → Upload documents (optional) → AI suggests skills → Review/edit skills → Set capture mode → Campaign created
```

**Key actions:**
- Name the expert and set departure timeline
- Upload existing docs (AI extracts skills automatically)
- Accept/reject AI-suggested skills
- Choose capture mode: Human-led / AI-guided / Hybrid

---

## 2. Plan Sessions

```
Open campaign → View suggested schedule → Adjust dates/times → Assign skills to sessions → Confirm schedule
```

**Key actions:**
- Review AI-recommended cadence (e.g., weekly for 6 weeks)
- Drag/drop skills into session slots
- Set session duration and participants

---

## 3. Conduct Capture Session

```
Start session → Record audio/video → Real-time transcription → AI provides guidance → Flag key insights → End session
```

**Key actions:**
- Start/pause/resume recording
- View AI coach suggestions (questions, topics to cover)
- Mark important moments manually
- See skill coverage progress in real-time

---

## 4. Review & Export

```
Session ends → Auto-generate transcript → AI extracts insights → Update knowledge graph → Generate reports → Export
```

**Key actions:**
- Review session summary
- Verify extracted skills and insights
- Explore knowledge graph connections
- Download reports (transcript, summary, graph export)

---

## 5. Monitor Progress (Dashboard)

```
View all campaigns → Check health status → Act on tasks → Review metrics → Get AI recommendations
```

**Key actions:**
- See campaign cards with progress indicators
- Triage tasks (urgent / this-week / on-track)
- Drill into at-risk campaigns
- Track coverage % and velocity metrics

---

## Flow Dependencies

```
Create Campaign → Plan Sessions → Conduct Sessions → Review/Export
                                        ↓
                              Dashboard (monitors all)
```

Each session builds on previous ones. Knowledge graph grows cumulatively across sessions.
